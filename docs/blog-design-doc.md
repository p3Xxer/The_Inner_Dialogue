# Blog Design Document

## Content Type

- Personal essays and opinions
- Book reviews / library

## Inspiration

- [leerob.io](https://leerob.io) — minimal, clean, short intro + post list, strong typography
- [cassidoo.co](https://cassidoo.co) — playful personality, friendly rounded font, tag-based post filtering
- [campedersen.com](https://campedersen.com/singularity) — creative, long-form essays, bold typography

## Additional Inspiration

- [brianlovin.com](https://brianlovin.com) — dark theme, clean layout, beautifully crafted details
- [manuelmoreale.com](https://manuelmoreale.com) — ultra-minimal, text-only, personal essays, zero decoration
- [paco.me](https://paco.me) — dark, minimal, smooth transitions, great typography
- [macwright.com](https://macwright.com) — clean, minimal, personal essays + technical writing, light/dark toggle

## Design Principles

- Minimal — let the writing be the focus
- No hero images — text-only post headers
- Dark theme as default
- Typography does the heavy lifting, not decoration

## Layout

- Single column, centered
- Content width: ~650–700px (optimal for essay readability)
- Top nav: name/logo + links (home, blog, about)
- No sidebar

## Pages

- **Home** — short intro (2–3 lines), recent posts list (title, date, one-line description)
- **Blog post** — title, date, reading time, body text
- **About** — who you are, what you write about, contact info
- **Library** (optional) — dedicated section for book reviews, separate from essays

## Typography

Target: warm, readable sans-serif that works well for long-form content.

| Font | Vibe | Link |
|---|---|---|
| Jost | Warm, rounded, closest to cassidoo feel | [fonts.google.com/specimen/Jost](https://fonts.google.com/specimen/Jost) |
| DM Sans | Clean geometric, modern | [fonts.google.com/specimen/DM+Sans](https://fonts.google.com/specimen/DM+Sans) |
| Inter | Neutral workhorse, very readable | [fonts.google.com/specimen/Inter](https://fonts.google.com/specimen/Inter) |
| Geist Sans | What leerob uses, Vercel's font | [vercel.com/font](https://vercel.com/font) |

- Body line-height: ~1.7
- Good paragraph spacing for essay readability

## Color

- Dark background: `#0a0a0a` or similar near-black
- Body text: `#e5e5e5` (soft white, easy on the eyes)
- One accent color for links and highlights (pick based on personality)
- Muted secondary text for dates, metadata: `#a3a3a3`

## Post List Style

- Title (linked)
- Date + optional reading time
- One-line description or excerpt
- Optional tags (like cassidoo.co)

## Post Page Style

- Title
- Date + reading time
- Clean body text, no hero image
- Minimal metadata, straight into content
